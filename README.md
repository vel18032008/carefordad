# CareForDad ❤️

**A Smart Health Companion for Parents with Diabetes, BP, and Chronic Conditions**

Built for the NIAT #BuildForDad Father's Day Challenge.

## 🔗 Live App
https://vel18032008.github.io/carefordad/

## 💡 The Problem

Parents managing chronic conditions like Type 2 Diabetes and Hypertension face daily challenges:
- Forgetting to take medicines on time
- No simple way for children living away to know if medicine was taken
- BP and sugar readings scattered with no trend tracking
- No alert system if something seems wrong

## ✅ The Solution

CareForDad connects **Parent → Guardian (child)** in one simple app:

- **Medicine reminders & confirmation** — parent taps "Taken" to confirm each dose
- **Smart escalation system** — if a dose isn't confirmed, reminders repeat, then the guardian is automatically notified
- **Health tracking** — BP and blood sugar logging with trend charts
- **Family dashboard** — guardian sees today's status, alerts, and adherence at a glance
- **AI weekly summary** — plain-language health summary generated from real adherence and reading data
- **CareBand (simulated wearable)** — demonstrates future heart-rate monitoring with automatic alerts on abnormal readings

## 🛠️ Tech Stack

- HTML, CSS, JavaScript (single-file app)
- Chart.js for BP/sugar/heart-rate trend graphs
- Tabler Icons
- Designed to integrate with Firebase (auth, Firestore, push notifications) and the Anthropic/OpenAI API for production AI summaries
